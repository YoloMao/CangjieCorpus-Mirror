# log 模块

## log 功能介绍

log 模块提供日志管理和日志打印接口。

## log 模块的包列表

log 模块提供了如下包：

|                              包名                              |    功能    |
| -------------------------------------------------------------- | --------- |
| [log](./log/log_package_overview.md)                 | log 包提供了一个单一的日志API，它抽象了实际的日志实现。 |
