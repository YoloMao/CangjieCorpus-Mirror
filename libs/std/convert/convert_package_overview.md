# std.convert 包

## 功能介绍

convert 包提供从字符串转到特定类型的 Convert 系列函数。

例如字符串 "true" 到布尔类型 `true` 的转换。

## API 列表

### 接口

|              接口名          |           功能           |
| --------------------------- | ------------------------ |
| [Parsable\<T>](./convert_package_api/convert_package_interfaces.md#interface-parsablet) | 将字符串解析为特定类型的接口。 |
